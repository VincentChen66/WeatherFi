WeatherFi 

This is my cs50's final project. cs50 is a Computer Science course (on-campus and online) offered by Harvard University. cs50 is quite popular among CS courses. My project name is "WeatherFi" where users can search for weather reports of any city around the globe. WeatherFi is licensed under the MIT license. This project is developed using Python, JavaScript, HTML, CSS, Flask and Bootstrap framework (version 4.5 and 5) for grids and containers. As per Github's estimation, WeatherFi uses 27.1% of HTML, 4.7% of Python, 54.7% of CSS, and 13.5% of JavaScript.
WeatherFi is up on the Internet leveraging the power of AWS. This project is licensed under MIT license, so there are very few restrictions.
